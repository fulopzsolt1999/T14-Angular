A Menus feladatok angular projekt lépéseinek leírása:

  1. Létehozom az angular projektet: ng new Menusfeladatok --no-standalone
    • Elkészítem a kezdőlap komponenst: ng g c kezdolap
    • Elkészítem a hibakezelő komponenst: ng g c hiba404
    • Elkészítem a téglalap komponenst: ng g c teglalap
    • Elkészítem a páros-páratlan komponenst: ng g c parosparatlan
    • Elkészítem a víz halmazállapota komponenst: ng g c halmazallapot
  2. Hiba404 oldal tartalmát elkészítettem: <h1 class="text-center">Hiba 404, a keresett oldal nem található</h1>
  3. Routing beállítása: app-routing.module.ts
    • Kezdőlap routingja:path: "kezdolap", component: KezdolapComponent
    • Hibakezelő routingja:path: "hiba", component: Hiba404Component
    • Téglalap komponens routingja:path: "teglalap", component: TeglalapComponent
    • Páros páratlan komponens routingja:path: "parosparatlan", component: ParosparatlanComponent
    • Halmazállapot komponens routingja: path: "halmazallapot", component: HalmazallapotComponent
    • Routing üres útvonal esetén: path: "", redirectTo: "/kezdolap", pathMatch: "full"
    • Routing minden egyéb esetben (ELSE ÁG) path: "**", component: Hiba404Component
  4. app.component.html tartamát törlöm kivéve a <router-outlet /> rész
  5. Elkészítek pár egyszerű linket a routing elérésekhez:
    • Kezdőlap komponenshez: <a routerLink="/kezdolap">Kezdőlap</a>
    • Téglalap komponenshez: <a routerLink="/teglalap">Téglalap</a>
    • Páros-páratlan komponenshez: <a routerLink="/parosparatlan">Páros páratlan</a>
    • Halmazállapot komponenshez: <a routerLink="/halmazallapot">Halmazállapot</a>
    • Hiba komponenshez:<a routerLink="/hiba">Hiba404</a>
  6. Telepítem a bootstrap-et: npm install bootstrap
    1. Módosítom az angular.json fájl tartalmát, a bootstrap elérési útvonalával
    2. Bootstrap CSS fájl útvonala:"styles": [ "src/styles.css","./node_modules/bootstrap/dist/css/bootstrap.css" ]
    3. Bootstrap JS fájl útvonala:"scripts": [ "./node_modules/bootstrap/dist/js/bootstrap.js" ]
    4. Tesztként futtatom az angular szervert: ng serve
    5. A linkeket elláttam megfelelő bootstrap osztályokkal: btn (btn-(primary-success-warning-info-danger)) m-2
  7. Importálom a Formsmodule-t mivel a feladatoknál űrlapokat szeretnék használni
  8. Elkészítem a téglalap komponenst:
    1. aOldal: number = 1; teglalap.component.ts állomány új mezője
    2. bOldal: number = 1; teglalap.component.ts állomány új mezője
    3. Elkészítem az űrlapot és a számításokat a teglalap.component.hmtl állományban
      • Űrlap elemek:
        <label for="aOldal">Az a oldal mérete:</label>
        <input type="number" id="aOldal" [(ngModel)]="aOldal" class="form-control">
        <label for="bOldal">Az a oldal mérete:</label>
        <input type="number" id="bOldal" [(ngModel)]="bOldal" class="form-control">
      • Interpolációs megjelenítés: {{' <p>A téglalap [A] oldalának mérete: {{aOldal}}</p> <p>A téglalap [B] oldalának mérete: {{bOldal}}</p> <p>A téglalap kerülete: {{2*(aOldal+bOldal)}}</p> <p>A téglalap területe: {{aOldal*bOldal}}</p> '}}
  9. Elkészítem a Párospáratlan komponenst
  10. Elkészítem a Halmazállapot komponenst
